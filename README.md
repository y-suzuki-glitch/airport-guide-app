# 空港アクセスガイド作成アプリ (BAIYotsugi2chome101様用)

成田空港・羽田空港から、お部屋（BAIYotsugi2chome101）までのルートガイド画像を、見本のようなハッキリとしたデザインで自動生成するアプリです。

## 🚀 使い方

### 1. 準備 (初回のみ)

* **Python のインストール**: Python 3 がインストールされていることを確認してください。
* **Macのフォント**: 画像生成にはMac標準の `Hiragino Sans`, `Apple Color Emoji` を使用します。これらがインストールされている必要があります。

### 2. ライブラリインストール

ターミナルでこのフォルダに移動し、以下のコマンドを実行します。

```bash
pip3 install -r requirements.txt
